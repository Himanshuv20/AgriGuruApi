API Overview

The AgriGuru API provides AI-powered crop recommendations and farming advice in multiple Indian languages. It combines advanced natural language processing, agricultural expertise, and multilingual translation capabilities to help farmers make informed decisions.

üåç Multilingual Support

Support for 11 Indian languages including Hindi, Bengali, Tamil, Telugu, and more.

  • Automatic language detection
  • Bidirectional translation
  • Native script support
  • Fallback translation system

üå± Smart Crop Recommendations

AI-powered recommendations based on location, soil, season, and farming conditions.

  • Contextual crop suggestions
  • Suitability scoring
  • Water requirement analysis
  • Economic viability assessment

üöÄ High Performance

Optimized for speed and reliability with advanced caching and error handling.

  • Response caching
  • Rate limiting protection
  • Graceful error handling
  • Sub-second response times

üìä Rich Context Understanding

Comprehensive understanding of Indian farming contexts and regional variations.

  • 8 soil type classifications
  • 3 cropping seasons (Kharif, Rabi, Zaid)
  • Multiple irrigation methods
  • Experience-based recommendations

üîí Authentication

Currently, the API does not require authentication for basic usage. Rate limiting is applied per IP address to ensure fair usage.

üìù Rate Limiting

The API implements rate limiting to ensure service availability:

  • Rate Limit: 100 requests per 15 minutes per IP
  • Headers: Rate limit information is provided in response headers
  • Reset: Rate limits reset every 15 minutes

üéØ Response Format

All API responses follow a consistent JSON format:

{ "success": true, "data": { /* response data */ }, "timestamp": "2025-07-12T10:30:00.000Z", "processingTime": "1250ms" }

Code Examples

üü® JavaScript (Fetch API)

const getCropAdvice = async () => { try { const response = await fetch('http://localhost:4000/api/v1/crop-advice', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ question: 'What crop should I grow in black soil during monsoon?', language: 'en', farmerContext: { location: 'Maharashtra', soilType: 'black', season: 'kharif', farmSize: '5 acres' } }) }); const data = await response.json(); console.log('Recommendations:', data.data.recommendations); } catch (error) { console.error('Error:', error); } };

üêç Python (requests)

import requests import json def get_crop_advice(): url = "http://localhost:4000/api/v1/crop-advice" payload = { "question": "What crop should I grow in black soil during monsoon?", "language": "en", "farmerContext": { "location": "Maharashtra", "soilType": "black", "season": "kharif", "farmSize": "5 acres" } } headers = { "Content-Type": "application/json" } try: response = requests.post(url, json=payload, headers=headers) response.raise_for_status() data = response.json() print("Recommendations:", data['data']['recommendations']) except requests.exceptions.RequestException as e: print(f"Error: {e}") get_crop_advice()
